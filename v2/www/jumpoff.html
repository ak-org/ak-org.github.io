Poverty and Heart disease mortality rate in United States

According to this article poverty is both a cause and consequence of poor health.

Following charts shows mortality rate amongst men and women in USA. The mortality rate is per 100,000 Adults. Men carries higher risk in majority of the states in USA. These charts corraborate the fact women in general carry less risk of heart disease as compared to men. The variance in women's chart implies that there are potentially other factors influencing the mortality rate among women.

Is there a relationship between the poverty and heart disease mortality rate? Do non-economic factors such as race, ethnicity, location or weather contribute to the high mortality rate?

The project follows the Martini Glass narrative visualization structure. This page represents the "jumping off point" of the visualization structure. The user has experienced author driven contents and is now ready for the reader driven content. The next and Final page contains reader driven content and it allows users to interact with data and draw their own conclusions.

User Interface Events

The buttons in the navigation bar at top and the Prev, Next button at the bottom of this page enable the User Interface events. User interacts by clicking on the button or menu items to navigate to various part of this story. Mouse hover events in the bar charts show additonal data details.

Annotations

Both charts are annotated via tooltips. These tooltips are visible when user hovers the mouse over these charts.

Triggers

The mouse click acts as a trigger event. The mouse hover acts a trigger event for tooltips in both charts.

Parameters

This section is hyper linked to the previous and next section. The name of the target section is passed as an input paramter to the buttons and navgiation bar items. The data is passed as a JSON object to Vega-lite javascript object for rendering axis and bars in the charts.
